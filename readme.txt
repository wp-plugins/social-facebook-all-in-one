== Description ==
	
Social Facebook All In One is a widget which allows to display , on your wordpress website , 
a customizable jQuery accordion that brings together different social plugins published by Facebook , 
that is to say :

* Facebook fans like box
* Facebook Activity Feed
* Facebook Recommandations
* Facebook Registration


Social Facebook All In One widget is distinguished by a prefix to avoid conflicts 
in database Options table .
Besides , it is based on a jQuery accordion which is animated by the following effects , for now :

* Bounceslide effect ( 12/15/2011 ) .

== Installation ==	

[DOCUMENTATION DOWNLOAD](http://donisocial.donimedia-servicetique.net/?wpfb_dl=2 "DOCUMENTATION DOWNLOAD") 

WIDGET INSTALLATION PROCEDURE :
 
1. WIDGET INSTALLATION :

1. Download Social Facebook All In One widget to your computer , from official Wordpress directory .
2. Launch your web browser and connect you to your Wordpress blog .
3. Click on the plugins menu .
4. Click on the Add New button .
5. Click on the Upload link .
6. Use the  browse  button to select the plugin zip file that was downloaded .
7. Click on the  Install Now  button .
8. Once the widget is installed, click on the Activate plugin button .
9. The widget is now ready to use !

 
2. WIDGET CONFIGURATION :

1. You can access Social Facebook All In One widget from  Appearance --> Widget menu or from widgets option of your theme .
2. Drag and drop  Social Facebook All In One widget towards the appropriate sidebar .
3. Enter a title for this widget .
4. Translate the predefined labels , at your convenience .
5. Enter your Facebook Url Page .
6. Enter your Facebook App Id . ( You will find a tutorial explaining how to retrieve a Facebook App id , also called client id , at the following address : http://donisocial.donimedia-servicetique.net/?p=49 )
7. The most important of remaining parameters,  is the "width" one. Whether the social blocks plugins, the registration box or the footer Like button,  be sure to choose an appropriate value for the width of your template sidebar .
8. Once your web page is refreshed , you can see that Social Facebook All In One widget is ready to use !
